# ProTask – MERN Task Management Application

## 📌 Overview

ProTask is a full-stack task management application built using the MERN stack. It enables users to manage their daily tasks efficiently through secure authentication and an intuitive interface. Users can create, update, delete, and track tasks while organizing their workflow effectively.

## 🚀 Features

* User authentication and authorization using JWT.
* Create, update, delete, and manage tasks (CRUD operations).
* Track task status (Pending, In Progress, Completed).
* Responsive and user-friendly interface.
* RESTful API integration between frontend and backend.
* Secure data storage using MongoDB.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/protask.git
```

### Navigate to the project

```bash
cd protask
```

### Install dependencies

Frontend:

```bash
cd client
npm install
npm run dev
```

Backend:

```bash
cd server
npm install
npm start
```

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
